# Neon Typing (寿司打風タイピングゲーム)

ブラウザで遊べる、モダンでリッチなUIを備えた本格的な日本語タイピングゲームです。
すべての処理がクライアントサイド（SPA）で完結しており、Cloudflare PagesやNginxを用いた静的ホスティング環境などに簡単にデプロイできます。

## 🌟 特徴

- **柔軟なローマ字入力対応**: 「し (shi / si / ci)」「ん (nn / n)」「っ (xtsu / ltu / 子音重ね)」など、プレイヤーのタイピングのクセに合わせたあらゆる入力方式（ヘボン式・訓令式・ワープロ式）に対応。
- **3段階の難易度**: お手軽・普通・高級の3つの難易度を選択可能。難易度ごとに出現する単語が変化します。
- **スコアシステム**: 正しく打ったキーの数、平均キータイプ数 (KPS: Keys Per Second)、ミスタイプ数、基本スコアをリアルタイムで計測します。
- **モダンなUI/UX**: グラスモーフィズムを取り入れた、直感的で美しいデザイン。画面サイズに応じた自動文字サイズ調整にも対応。
- **途中終了機能**: タイピング中に `Esc` キーを押すことで、その時点のスコアでゲームを終了しリザルト画面へ移行できます。

## 🛠️ 技術スタック

- **フレームワーク**: React (Vite)
- **言語**: TypeScript
- **スタイリング**: Vanilla CSS (CSS Variables)
- **デプロイ**: 静的ファイル (HTML/CSS/JS) として出力されるため、任意のWebサーバーでホスト可能。

## 🚀 ローカルでの動かし方

Node.js がインストールされている環境で、以下のコマンドを実行してください。

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:5173/` にアクセスするとゲームをプレイできます。

## 📝 カスタマイズ（仕様）

出題される単語や難易度の設定は `src/data/config.ts` で管理されています。

- `DIFFICULTIES`: 難易度のID、名前、制限時間（秒）を定義。
- `WORDS`: 各難易度グループに属する単語とその読み方（ひらがな）を定義。

※「普通」を選択した場合は「お手軽＋普通」の単語が、「高級」を選択した場合は「普通＋高級」の単語がミックスされて出題されるロジックになっています。

## 🌐 デプロイについて

本アプリケーションはサーバーサイドのロジックを持たない静的SPAです。

```bash
npm run build
```
を実行すると `dist` フォルダが生成されます。この中身をそのままサーバー（Nginxなど）の公開ディレクトリに配置するか、Cloudflare Pages等にアップロードするだけでデプロイが完了します。
